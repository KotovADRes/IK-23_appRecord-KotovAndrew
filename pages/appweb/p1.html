ТЕХНОЛОГІЇ

  • HTML: Основа веб-сторінок, що використовується для структурування контенту. Визначає заголовки, абзаци, списки, зображення та інші елементи на сторінці.
  • CSS: Мова для опису зовнішнього вигляду HTML-документів. Використовується для:
    • Базове стилізування: Застосування шрифтів, налаштування відступів (margins/paddings), стилізація списків та заголовків.
    • Макетування:
      • Flexbox: Модуль для створення гнучких макетів в одному вимірі, що дозволяє легко вирівнювати та розподіляти простір між елементами в контейнері.
      • Position Sticky: Властивість позиціонування, що дозволяє елементу "прилипати" до певної позиції у вьюпорті під час прокручування сторінки. Використовується у хедері
      • Гнучка блочна верстка: Загальний підхід до створення адаптивних макетів, які добре виглядають на різних розмірах екранів.
    • Інтерактивність та анімації:
      • Hover-ефекти: Зміна стилів елемента при наведенні на нього курсора миші.
      • CSS Transitions: Плавні переходи між різними станами стилів елемента, наприклад, для анімації появи/зникнення (наприклад, "хрестик" в бургер-меню).
  • JavaScript (JS): Мова програмування, що використовується для додавання інтерактивності та динамічного функціоналу на веб-сторінки. У даному випадку застосована для реалізації функціональності бургер-меню (відкриття/закриття меню при кліку на іконку).
