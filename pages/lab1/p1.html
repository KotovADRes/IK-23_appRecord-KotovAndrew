ОПИС ПРЕДМЕТНОГО СЕРЕДОВИЩА

Розроблюваний програмний продукт є музичним сервісом, що надає користувачам доступ до аудіоконтенту на певних умовах.

Ключовою особливістю сервісу є вимога обов'язкової авторизації для будь-якої взаємодії з платформою, зокрема для прослуховування треків. Без успішного входу в систему користувачі не матимуть можливості відтворювати музику.

Контент сервісу поділяється на дві основні категорії: безкоштовні та платні треки. Безкоштовні треки доступні для прослуховування всім зареєстрованим та авторизованим користувачам без додаткових умов. Платні треки, у свою чергу, вимагають наявності активної підписки та мають два рівні доступу, що визначають, який саме контент буде доступний користувачеві.

Для доступу до платних треків передбачено систему підписок. Існує два типи підписок:

  • Підписка А: відкриває доступ до треків рівня А.
  • Підписка B: надає ширший доступ, охоплюючи треки як рівня А, так і рівня B.

Важливо зазначити, що будь-яка підписка має обмежений термін дії та потребує періодичного подовження для збереження доступу до відповідного платного контенту.

Процес прослуховування треків чітко регламентований. Авторизовані користувачі можуть безперешкодно слухати всі безкоштовні композиції. Для відтворення платних треків необхідно мати активну підписку, що відповідає рівню доступу конкретного треку. Якщо користувач спробує прослухати трек, до якого у нього немає доступу через відсутність або невідповідність підписки, система запропонує йому оформити або оновити підписку.

Кожен зареєстрований користувач має особистий кабінет. У цьому розділі сервісу користувачі можуть:

  • Переглядати інформацію про свої поточні підписки.
  • Керувати ними (наприклад, подовжувати або змінювати тип підписки).
  • Редагувати свої особисті дані, надані при реєстрації.

Для адміністрування сервісу передбачена спеціальна адмін-панель. Інструменти адміністраторів для управління музичним контентом включають можливість додавання нових треків та видалення існуючих. Також вони відповідають за призначення рівня доступу (безкоштовний, рівень А, рівень B) для кожного треку. Окрім управління контентом, адмін-панель надає функціонал для керування користувачами, їхніми обліковими записами та підписками, забезпечуючи належне функціонування та підтримку сервісу.
